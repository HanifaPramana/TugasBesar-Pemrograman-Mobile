# 7. Berikut tampilan menu detail dari daftar nama tempat Kuliner di Malang, kami ambil contoh Ronde Titoni
<img src="https://github.com/syaikhulalim/TugasBesar-Pemrograman-Mobile/blob/master/image/11.jpeg" width="291" height="600">
<hr>
<img src="https://github.com/syaikhulalim/TugasBesar-Pemrograman-Mobile/blob/master/image/12.jpeg" width="291" height="600">

Pada menu detail dari Ronde Titoni, terdapat informasi-informasi seperti Alamat, Jam Operasional, Range Harga, Deskripsi, Kontak dari tempat Kuliner Ronde Titoni. <br> 
Pada bagian atas, kami menggunakan fungsi Picasso. Yakni fungsi dimana kami mengambil gambar untuk mengisi field dari Internet. <br> 
Lalu terdapat fitur yakni ‘Lihat Peta Lokasi’ dengan memanfaatkan API google maps, Longitude, Altitude.
